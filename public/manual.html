1. Introducción

El Validador de Documentos Excel es una herramienta web especializada para validar archivos Excel destinados a comunicaciones masivas...

2. Requisitos del Sistema

  • Navegadores compatibles: Chrome, Firefox, Edge, Safari
  • Formatos soportados: .xlsx, .xls
  • Tamaño máximo: 10 MB
  • Conexión a internet: Requerida

3. Acceso a la Aplicación

  1. Abrir navegador web
  2. Navegar a la URL de la aplicación
  3. Cargar la página principal

4. Interfaz de Usuario

Descripción de cada área: header, área de carga, resultados, footer.

5. Campos de Validación

Se validan campos obligatorios como tratamiento, nombres, departamento y municipio. También se aceptan campos opcionales como cargo, entidad, etc.

6. Proceso de Validación

  1. Preparar archivo Excel
  2. Cargar el archivo
  3. Ejecutar validación
  4. Revisar resultados

7. Interpretación de Resultados

Errores en rojo (críticos), advertencias en amarillo. Vista por lista, campo o fila.

8. Corrección de Errores

Corregir primero los errores críticos. Validación iterativa recomendada.

9. Plantilla Excel

Puede descargar una plantilla que incluye encabezados y ejemplos válidos.

10. Preguntas Frecuentes

Respuestas sobre tamaño de archivos, nombres de columnas, validación masiva, etc.

11. Solución de Problemas

Guía rápida para solucionar errores comunes durante la validación.
